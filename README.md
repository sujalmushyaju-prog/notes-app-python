# 📝 Notes App (Python)

A simple command-line Notes App built using Python.  
This project allows users to add, view, edit, and delete notes.

---

## 🚀 Features

- Add notes  
- View notes  
- Edit notes  
- Delete notes  
- Clear all notes  
- Notes are saved in a file  

---

## ▶️ How to Run

1. Install Python  
2. Download the project  
3. Run this command:

python notes_app.py
